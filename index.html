What this tool does ▶
  • Extracts patent publication numbers from supported STN fields and formats jurisdiction specific output suitable to be copied and pasted to EPO AbS.
  • Extracts PubMed identifiers for Medline and Embase output blocks.
  • Extracts WPIDS accession numbers and formats them for STN search syntax.
  • Preserves the original extraction logic and live update behaviour from the source tool.
Output panels ▶
Patents-EPO AbS Medline Embase WPIDS
  • Outputs are produced in STN-ready bracketed syntax.
  • Patent results are de-duplicated and sorted for readability.
  • Each output panel includes a compact one-click copy to clipboard control.
